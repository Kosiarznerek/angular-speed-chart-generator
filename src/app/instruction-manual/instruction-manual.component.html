Instruction manual

  1. Prepare two files 'distances.txt' and 'times.txt'
  2. Each of file should contain input data as follow

    A
    B
    C
    ...
    X
    Y
    Z

    Where A,B,C,... are number values separated from each other by new line.
  3. 'distances.txt' is a text file containing traveled distance after corresponding in 'times.txt' file time.
  4. To generate charts go to 'main-view' tab and drag both your 'distances.txt' and 'times.txt' files to drag section.
  5. You can download example files below.
